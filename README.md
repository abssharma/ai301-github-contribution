## Understanding the Issue

### Problem Description

This issue focuses on improving Copier's documentation around template preview workflows. During the discussion, a maintainer recommended the third-party tool [`ctt`](https://github.com/copier-org/copier/issues/1451#issuecomment-1859240085) for snapshot-based testing and template preview. However, another contributor pointed out that this workflow is not currently easy to [`discover`](https://github.com/copier-org/copier/issues/1451#issuecomment-1861863200) through the documentation. As a result, template authors looking for preview functionality may assume that Copier does not provide a recommended solution. The goal of this issue is to make the recommended workflow easier for users to find and understand.
